# Portfólio Mateus Maziero

Este repositório contém o portfólio pessoal de Mateus Maziero em um único arquivo HTML responsivo e moderno.

## Sobre o projeto
- Página de apresentação pessoal com seção de destaque, sobre, stack técnica, projetos e contato.
- Design escuro, tipografia refinada e animações sutis de entrada.
- Feito com HTML e CSS puros.

## Conteúdo
- `index.html` — entrada principal do portfólio.
- `portfolio_mateus_maziero_v3.html` — arquivo do portfólio.

## Como visualizar
1. Abra `index.html` em um navegador.
2. Ou use um servidor local para desenvolvimento:
   - Python 3:
     ```bash
     python -m http.server 8000
     ```
   - Depois, acesse `http://localhost:8000`.

## Publicar no GitHub Pages
1. Inicialize o repositório Git:
   ```bash
   git init
   git add .
   git commit -m "Inicializa portfólio"
   ```
2. Crie um repositório no GitHub e adicione o remoto:
   ```bash
   git remote add origin https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
   git branch -M main
   git push -u origin main
   ```
3. Ative o GitHub Pages nas configurações do repositório:
   - Vá em **Settings > Pages**
   - Escolha a branch `main` e a pasta `/ (root)`
   - Salve e acesse a URL fornecida.

## Observação
Substitua `SEU_USUARIO` e `SEU_REPOSITORIO` pelo nome do seu usuário e repositório no GitHub.
